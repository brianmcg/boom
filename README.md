# Boom

Boom is a first person shooter built using [PixiJS](http://www.pixijs.com/) + [HOWLER.JS](https://howlerjs.com/).

## Installation

Use Node.js v18.14.1

```bash
npm install
```

## Serve locally

```bash
npm start
```

## Build

```bash
npm run build
```
