# Boom

Boom is a first person shooter built using [PixiJS](http://www.pixijs.com/) + [HOWLER.JS](https://howlerjs.com/).

## Installation

Use Node.js v22.11.0

```bash
nvm use
```

```bash
npm install
```

## Serve locally

```bash
npm run dev
```

## Build

```bash
npm run build
```
